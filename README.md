# Домашнее задание к занятию "7. Работа с HTTP"
### Система управления заявками (HelpDesk)

[![Netology AHJ](https://github.com/Milka79rus/ahj-http-helpdesk/actions/workflows/web.yml/badge.svg)](https://github.com/Milka79rus/ahj-http-helpdesk/actions/workflows/web.yml)

## Описание проекта
Frontend-часть приложения "HelpDesk" для управления тикетами (заявками). Приложение позволяет просматривать, создавать, редактировать и удалять тикеты, а также изменять их статус.

**Функционал:**
* Получение списка тикетов с сервера.
* Просмотр подробного описания конкретного тикета.
* Модальные окна для добавления и удаления.
* Управление статусом тикета (выполнено / не выполнено).

## Технологии
* HTML, CSS, JavaScript (ES6+)
* Webpack (сборка проекта)
* GitHub Actions (CI/CD: проверка линтером, тесты и автоматический деплой)

## 🛠 Инструкция для проверки
Так как проект использует бэкенд-сервер, для корректной работы необходимо:
1. Клонировать репозиторий.
2. Перейти в папку с сервером (если он отдельно) и запустить его: `npm run start` (или ваша команда запуска).
3. В другой консоли перейти в папку `frontend`, установить зависимости и запустить фронтенд: `npm install && npm run start`.
4. Приложение будет доступно на `http://localhost:8080`.
